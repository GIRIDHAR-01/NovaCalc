# NovaCalc

A modern desktop calculator built with Python and PySide6.

## Features

- Standard Calculator
- Scientific Calculator
- Clean Modern UI
- Dark Theme
- Keyboard Support
- Fast Performance

## Installation

```bash
git clone https://github.com/GIRIDHAR-01/NovaCalc.git
cd NovaCalc
pip install -r requirements.txt
python app.py
```

- Python
- PySide6
- Qt
