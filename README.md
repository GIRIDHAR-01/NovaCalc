# NovaCalc

A modern desktop calculator built with Python and PySide6.

## ✨ Features

- Standard Calculator
- Scientific Calculator
- Programmer Mode
- Matrix Calculator
- Statistics Calculator
- Unit Converter
- Workspace Notes
- Calculator History
- Modern Dark UI
- Smooth Animations

## 🚀 Installation

```bash

cd NovaCalc

pip install -r requirements.txt

python main.py
```
